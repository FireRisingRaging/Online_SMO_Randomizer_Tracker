SMO Randomizer Tracker

ℹ️ OBS Setup

Popup Window Recommended

Click Open OBS Overlay to open a 315×450 popup then capture it with Window Capture in OBS.

Browser Source

  1. In OBS add a Browser Source
  2. Set URL to https://firerisingraging.github.io/Online_SMO_Randomizer_Tracker/obs.html
  3. Set width 315, height 450
  4. Enable Shutdown source when not visible
  5. Open the main tracker in the same browser: state syncs automatically

Removing the Background

  1. Click Toggle OBS BG in the tracker (or the BG button in the overlay corner)
  2. In OBS right-click the source → Filters
  3. Add a Chroma Key filter
  4. Match the key color to OBS BG Color in Settings (default #00FF00)
