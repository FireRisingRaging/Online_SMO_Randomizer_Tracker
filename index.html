ℹ️ OBS Setup

Browser Source Recommended

  1. Enter or generate a Room Code in the tracker, then click Connect
  2. Copy the OBS URL that appears
  3. In OBS add a Browser Source
  4. Paste the OBS URL. Default size is 945 × 1350
  5. To make the overlay larger, change Popup Scale in Settings. The Browser Source Scale is always 3x that value, and the sync section shows the calculated width and height to paste into OBS.
  6. The overlay background is transparent so no chroma key needed

Popup Window

Click Open Popup to open a window with a dark background sized to your Popup Scale, then capture it with Window Capture in OBS. You can add a color key in the format options to remove the dark gray background(#181818)
